Instructions

Guide Nessie and her giant tail around the loch using arrow keys. Dive using "d" and surface using "s", but be careful not to run out of air. Avoid boats slamming into parts of your tail (ouch). Instead, eat boats for points and to relieve hunger. Eating boats makes Nessie grow longer, but biting one's tail spells game over.
